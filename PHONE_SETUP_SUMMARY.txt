🔧 MANUAL SETUP (If script doesn't work):

  1. Find your IP:
     hostname -I  (Linux)
     ipconfig     (Windows)
     
  2. Update mobile/src/utils/constants.ts:
     export const API_URL = 'http://YOUR_IP:5000/api/v1';
     
  3. Start backend:
     cd backend && npm run start:dev
     
  4. Start mobile:
     cd mobile && npm start
     
  5. Scan QR with Expo Go app

🐛 TROUBLESHOOTING:

  Can't connect?
  → Check same WiFi network
  → Verify IP in constants.ts
  → Test: curl http://YOUR_IP:5000/api/v1/health
  
  Firewall blocking? (Linux)
  → sudo ufw allow 5000
  → sudo ufw allow 8081
  → sudo ufw allow 19000:19001/tcp
