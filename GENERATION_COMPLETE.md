## 🚀 How to Use

### Quick Start (3 steps):

1. **Install dependencies**
   ```bash
   cd mobile
   npm install
   ```

2. **Start backend**
   ```bash
   cd ../backend
   npm run start:dev
   ```

3. **Start mobile app**
   ```bash
   cd mobile
   npm start
   ```

### Or use the automated script:
